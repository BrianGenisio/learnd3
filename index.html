    About LearnD3.com

    My name is Brian Genisio and I built this site to help me learn D3.js. I often find that learning new technologies works much better for me if I am committed to teach others about it. In the process, I am learning more about screencasting and communication. This site is a work in progress.

    D3.js is an amazing library designed to transform data into amazing visualizations. The learning curve is somewhat steep, but hopefully these screencasts will help people (as well as myself) learn the technology.

    The format for this site was insprired by John Lindquist's Egghead.io; a similar site developed for learning Angular.js. I found it to be so invaluable when learning Angular.js, that I thought I'd try the same for D3.js.
